# Collapsible Navigation Item

The collapsible navigation item component is used to simplify a navigation bar by allowing the user to reveal certain routes only if they need it.

## Example

<Canvas>
  <Story name="Primary">
    <div id="example-container" style={{ width: "25%" }}>
      <CollapsibleNavItem title="Lorem ipsum dolor sit amet" titleUrl="/">
        Inctum, laoreet enim a
      </CollapsibleNavItem>
    </div>
  </Story>
</Canvas>
